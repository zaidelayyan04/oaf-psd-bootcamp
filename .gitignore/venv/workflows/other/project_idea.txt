# Project Idea 
Fitness/Strength Progress Tracker

# Goal
The goal of this is to build a progresss tracker that can track and anaylze fitness progress 
over time. It will collect workout data, nutrition, and fitness goals and alayze them with charts.

# Data Source
User input- manual entry of excersises, sets, reps, and weights. Macronutrient calculator. 
User inputs SMART goals which are tested.

Possiby API- Any free apis of apps such as myfitnessplan.


# Data Storage
Local database SQLite for storing workouts
Possible Google sheets or excel for data backup

# Data Utilization
Progress tracking of weight lifted, number of workouts, and PRs.
Notifications for workouts
Visualization- charts that show progress over time. Could be weight, PRs, volume, ect.
